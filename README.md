# Syntaxwear - Tênis e Sneakers Online 👟

A **Syntaxwear** é uma landing page de e-commerce moderna e elegante, focada na venda de calçados exclusivos, como o destaque "Krypton One". O projeto foi desenvolvido com foco em design responsivo e uma experiência de usuário fluida.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as melhores práticas de desenvolvimento web front-end:

- **HTML5**: Estrutura semântica para melhor acessibilidade e SEO.
- **CSS3**: Estilização modular utilizando variáveis e metodologias modernas.
- **Google Fonts**: Integração das fontes 'Ubuntu', 'Poppins', 'Inter' e 'Roboto'.

---

## ✨ Funcionalidades

- **Design Responsivo**: O site se adapta perfeitamente a dispositivos móveis e desktops.
- **Menu Mobile**: Navegação otimizada para telas menores com menu hambúrguer.
- **Seção Hero**: Destaque de impacto para o produto principal.
- **Categorias Interativas**: Navegação rápida por estilos (Casual, Esporte, Moderno, Futurista).
- **Grid de Produtos**: Exposição visualmente atraente dos sneakers em destaque.
- **Footer Completo**: Inclui newsletter, links de redes sociais e navegação detalhada.

---

## 📁 Estrutura do Projeto

A organização dos arquivos segue uma estrutura modular para facilitar a manutenção:

```text
ecommerce-syntaxwear/
├── index.html          # Página principal
├── css/                # Estilos divididos por componentes
│   ├── variables.css   # Variáveis globais (cores, fontes)
│   ├── reset.css       # Reset de estilos padrão
│   ├── header.css      # Estilos do cabeçalho e menu
│   ├── hero.css        # Estilos da seção de destaque
│   ├── product-category.css # Estilos dos cards de categorias
│   ├── product-grid.css     # Estilos da grade de produtos
│   ├── footer.css      # Estilos do rodapé
│   └── style.css       # Estilos gerais e importações
├── images/             # Ativos visuais
│   ├── banners/        # Imagens de fundo (Hero)
│   ├── icons/          # Ícones de interface e redes sociais
│   ├── logo/           # Logotipo da marca
│   └── products/       # Fotos dos produtos (Sneakers)
└── README.md           # Documentação do projeto
```

## 🛠️ Como Visualizar o Projeto

Para ver o site funcionando na sua máquina local:

1. Clone este repositório ou baixe os arquivos.
2. Navegue até a pasta do projeto.
3. Abra o arquivo `index.html` em qualquer navegador moderno.
